## Portfolio Talking Points

- Designed a timestamp-based operational data model
- Built realistic synthetic data for 50 stores, 5,000 employees, and 12,000 orders
- Created fulfillment KPIs such as SLA %, pick rate, staging time, customer wait, substitution rate, and efficiency score
- Built a command-center dashboard with drill-through pages
- Modeled manager override authority as auditable actions instead of editable metrics
- Prepared data for MongoDB Atlas and Power BI
